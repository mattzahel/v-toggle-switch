# Customization

## Options

### value

-   Type: `boolean`
-   Default: `false`

### width

Button width in px

-   Type: `number`
-   Default: `50`

### height

Button height in px

-   Type: `number`
-   Default: `24`

### disabled

-   Type: `boolean`
-   Default: `false`

### outlineOff

Disables default focus outline. Not recommended!

-   Type: `boolean`
-   Default: `false`
