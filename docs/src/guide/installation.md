# Installation

Use `npm install v-toggle-switch` or `yarn add v-toggle-switch`

## Import

-   locally

```
import ToggleSwitch from 'v-toggle-switch'

export default {
    components: {
        ToggleSwitch
    }
}
```

-   or globally

```
import ToggleSwitch from 'v-toggle-switch'

Vue.use(ToggleSwitch)
```
